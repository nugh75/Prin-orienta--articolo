# IIPO

Indice sintetico che quantifica la completezza della documentazione delle pratiche di orientamento nel PTOF.

## Formula

$$IIPO = \frac{Media_{Strutturale} + Media_{Finalità} + Media_{Obiettivi} + Media_{Governance} + Media_{Didattica} + Media_{Opportunità}}{6}$$

Media aritmetica semplice dei punteggi delle 6 dimensioni del [[Framework di Valutazione a 6 Dimensioni|framework]] (scala Likert 1-7).

## Interpretazione

| Range | Livello |
|-------|---------|
| 0.0 – 1.0 | Non classificabile / Errore |
| 1.1 – 2.2 | Accenni minimi |
| 2.3 – 3.4 | Riferimenti generici |
| 3.5 – 4.6 | Azioni basilari |
| 4.7 – 5.8 | Sistema strutturato |
| 5.9 – 7.0 | Sistema eccellente |

## Limiti

- **Effetto compensazione**: la media può mascherare squilibri tra dimensioni
- **Sensibilità strumentale**: coglie variazioni incrementali nelle singole dimensioni
